# 🏘️ 社会工作者

> 社会工作者职业水平考试（社工证）备考

## 考试信息

| 级别 | 科目 | 报考条件 |
|-----|------|---------|
| 初级（助理社工师） | 综合能力+实务 | 高中/中专+4年工作经验 |
| 中级（社工师） | 综合能力+实务+法规政策 | 大专+6年工作经验 |
| 高级（高级社工师） | 笔试+评审 | 本科+10年工作经验 |

## 考试时间

- **报名：** 每年3-4月
- **考试：** 每年6月
