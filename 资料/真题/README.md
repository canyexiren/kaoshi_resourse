# 📄 历年真题汇总

> 🗂️ 本目录用于存放或链接历年考试真题

## 真题来源建议

### 线上刷题

- **粉笔APP：** 最全的真题库，免费刷
- **华图在线：** 真题+详细解析
- **中公题库：** 按知识点分类练习

### 下载渠道

- **国家公务员局** (scs.gov.cn) — 国考大纲及真题
- **各省人社厅官网** — 省考信息
- **B站** — 搜索"考试+年份+真题解析"
- **知乎/小红书** — 回忆版真题

## 真题使用指南

1. **限时做：** 严格按照考试时间
2. **复盘：** 花2倍于做题的时间复盘
3. **错题本：** 记录错题，定期回顾
4. **多次刷：** 真题至少刷3遍
