# 💻 考研专业课

> 专业课因专业而异，以下为通用备考建议

## 信息来源渠道

| 渠道 | 说明 |
|-----|------|
| 目标院校研究生院官网 | 查看参考书目和大纲 |
| 考研帮APP | 院校专业信息 |
| 各类考研论坛 | 经验分享 |
| 淘宝/闲鱼 | 购买学长学姐笔记 |

## 备考建议

1. **确定参考书：** 以目标院校指定的为准
2. **搜集真题：** 近5-10年真题，分析出题规律
3. **整理笔记：** 构建知识框架
4. **背书+做题：** 反复背诵专业知识点
