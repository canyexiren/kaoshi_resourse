## 备考规划

1. **基础（3-6月）：** 过教材+基础班
2. **强化（7-9月）：** 刷题+强化班
3. **真题（10-11月）：** 刷近15年真题
4. **冲刺（12月）：** 模拟卷+查漏补缺
