# 📖 考研英语

## 考试内容

| 题型 | 分值 | 说明 |
|-----|------|------|
| 完形填空 | 10分 | 20题，每题0.5分 |
| 阅读理解A | 40分 | 4篇，每篇5题 |
| 阅读理解B（新题型） | 10分 | 七选五/排序/小标题 |
| 翻译 | 10分（英一）/15分（英二） | 英译中 |
| 写作A（小作文） | 10分 | 书信/通知等 |
| 写作B（大作文） | 20分（英一）/15分（英二） | 图画/图表作文 |

## 备考规划

1. **单词（全程）：** 每天100-200个，反复背
2. **阅读（6-9月）：** 精研真题，逐篇分析
3. **写作（10-12月）：** 背模板+练笔
4. **全科（11-12月）：** 全真模拟
