## 考试信息

| 项目 | 国考 | 省考/联考 |
|-----|------|----------|
| 时间 | 每年11-12月笔试 | 每年3-4月笔试 |
| 科目 | 行测+申论 | 行测+申论（部分有专业课） |
| 报名 | 国家公务员局官网 | 各省人社厅官网 |
| 特点 | 竞争激烈，岗位层次高 | 岗位多，地域性强 |

## 备考策略

1. 先打基础（1-2个月）
2. 专项刷题（2-3个月）
3. 全真模考（1个月）
4. 考前冲刺（2周）

- **时政：** 学习强国、半月谈
