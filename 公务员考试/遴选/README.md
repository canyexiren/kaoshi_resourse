# 🎯 遴选

> 公务员遴选考试 — 基层公务员向上流动的重要通道

## 考试信息

| 项目 | 说明 |
|-----|------|
| **报考条件** | 已满服务期的在编公务员 |
| **考试科目** | 案例分析+对策性论文（A/B卷） |
| **考试时间** | 中央遴选每年11月，各省不同 |

## 考试内容

- **案例分析：** 机关实务、公文处理
- **对策性论文：** 政策理解+解决实际问题能力

## 备考建议

1. 注重日常积累，多看政策文件
2. 机关公文写作能力是核心
3. 关注中央和地方最新政策
