# 📝 申论

## 考试内容

- **归纳概括** — 提炼材料核心信息
- **综合分析** — 分析问题原因、影响等
- **提出对策** — 针对问题提出解决方案
- **公文写作** — 各类公文格式
- **大作文** — 议论文写作

## 素材积累

- 每日阅读：人民日报评论、半月谈
- 每周整理：时政热点、政策文件
- 每月总结：写作素材库

## 提分要点

1. 小题是基础，决定下限
2. 大作文是上限，决定高分
3. 字迹工整非常关键
4. 必须动手写，不能只看
