# 🎤 面试

## 面试形式

- **结构化面试** — 最常见形式
- **无领导小组讨论** — 小组讨论测评
- **结构化小组** — 结构化+互评

## 常见题型

| 题型 | 示例 | 答题思路 |
|-----|------|---------|
| 综合分析 | 社会现象、政策理解 | 现象-原因-影响-对策 |
| 组织管理 | 活动策划、调研 | 准备-实施-总结 |
| 应急应变 | 突发情况处理 | 冷静-分析-解决-反思 |
| 人际关系 | 同事/领导/群众矛盾 | 态度-沟通-解决-总结 |
| 岗位认知 | 你为什么报考 | 匹配度+职业规划 |

## 备考建议

1. 开口练是唯一捷径
2. 积累自己的例子库
3. 反模板，答出个人特色
4. 仪态：微笑+眼神交流+洪亮
