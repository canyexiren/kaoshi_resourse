# 📝 行测（行政职业能力测验）

## 考试内容

- **常识判断：** 政治、经济、法律、历史、文化、科技等
- **言语理解：** 阅读理解、逻辑填空、语句表达
- **数量关系：** 数学运算、数字推理
- **判断推理：** 图形推理、定义判断、类比推理、逻辑判断
- **资料分析：** 文字材料、表格材料、图形材料

## 刷题建议

1. 先分模块刷（各500题打底）
2. 再刷套卷（计时）
3. 资料分析 > 判断推理 > 言语 > 数量 > 常识（按优先级）
