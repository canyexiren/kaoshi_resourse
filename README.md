## 🎯 考试类型导航

| 考试类型 | 时间 | 特点 | 适合人群 |
|---------|------|------|---------|
| 🏛️ **国家公务员考试（国考）** | 每年11-12月 | 规模最大，竞争激烈 | 应届生为主，往届生也可 |
| 🏢 **省考/联考** | 每年3-4月（联考） | 各省组织，岗位多 | 所有符合条件者 |
| 🏛️ **事业单位统考** | 每年5月、10月 | 考试科目灵活 | 希望稳定编制者 |
| 👨‍🏫 **教师招聘** | 各地不同，多在上半年 | 需教师资格证 | 师范生及教育从业者 |
| 🎓 **全国考研** | 每年12月 | 学术深造 | 本科毕业生 |

## 💡 备考经验分享

### 📅 备考时间规划

| 阶段 | 时长 | 目标 |
|-----|------|------|
| 🌱 基础阶段 | 1-2个月 | 全面学习知识点 |
| 🌿 强化阶段 | 2-3个月 | 专项突破+大量刷题 |
| 🌳 冲刺阶段 | 1个月 | 全部真模考+查漏补缺 |

### 🎯 行测备考策略

1. **资料分析** 是拿分大户，必须练到100%正确率
2. **判断推理** 技巧性强，刷题见效最快
3. **言语理解** 靠积累，多读多练
4. **数量关系** 不要全放弃，掌握简单题型
5. **常识判断** 日常积累为主，考前突击时政

### 🎯 申论备考策略

1. **小题是得分关键** — 归纳概括要精准
2. **大作文重结构** — 标题+开头+分论点+结尾
3. **字迹工整** 很重要，影响3-5分
4. **日常积累** — 每天读人民日报评论
5. **一定要动笔写** — 只看不写等于白看

### 🎯 面试备考策略

1. **开口练** — 没有捷径，只有多练
2. **反模板** — 考官已经听腻了模板
3. **积累素材** — 准备自己的例子库
4. **模拟考场** — 找人对练，还原考场压力
5. **注意仪态** — 微笑、眼神交流、声音洪亮

- **知乎专栏** — 搜索"公考上岸经验"、"行测80+经验"
