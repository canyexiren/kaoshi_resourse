# 🏛️ 事业单位考试

> 事业单位统考、联考及各地市事业单位招聘备考资源

## 考试类型

| 类型 | 时间 | 科目 |
|-----|------|------|
| 事业单位联考 | 每年5月、10月 | 职测+综应 |
| 各省统考 | 各地不同 | 公基+职测/专业 |
| 单位单独招聘 | 全年 | 根据岗位定 |

## 考试科目

### 公共基础知识（公基）
- 政治理论（占比最大）
- 法律基础
- 经济常识
- 公文写作与处理
- 人文历史
- 科技环保

### 职业能力倾向测验（职测）
与公务员行测类似，增加策略选择题型

### 综合应用能力（综应）
类似申论，但岗位针对性更强

## 推荐名师

| 科目 | 名师 |
|-----|------|
| 公基 | 李梦娇（口诀记忆法） |
| 公基法律 | 孔令臣 |
| 职测 | 参考行测名师 |

## 备考建议

1. 公基重记忆，需要反复背诵
2. 职测重技巧，刷题提分快
3. 综应重逻辑，多看范文
