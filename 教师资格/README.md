## 教师招聘

### 考试内容

- **教育综合知识（教综）：** 教育学、心理学、教育法规
- **学科专业知识：** 所报学科的专业知识
- **公共基础知识：** 部分地区考

### 推荐资源

- 山香教育 — 教招老牌机构
- 粉笔教师APP — 刷题
- 超格教育 — 课程质量高

### 面试形式

- 试讲（无生上课）
- 说课
- 结构化面试
