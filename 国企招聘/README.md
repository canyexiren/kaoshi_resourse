# ⚡ 国企招聘

> 国家电网、烟草系统等国有企业招聘考试备考

## 考试类型

| 企业 | 考试内容 | 招聘时间 |
|-----|---------|---------|
| 国家电网 | 专业知识+行测+能源战略 | 每年11月、3月 |
| 烟草系统 | 行测+申论+专业知识 | 各地不同 |
| 中石化/中石油 | 行测+企业文化 | 每年9-10月 |
| 三大运营商 | 行测+企业文化 | 每年9-10月 |
| 银行校招 | EPI+英语+综合知识 | 每年9-10月 |
