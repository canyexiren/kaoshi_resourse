# 🪖 军队文职

> 军队文职人员招录考试备考资源

## 考试信息

| 项目 | 说明 |
|-----|------|
| **招聘时间** | 每年1月发布公告，2-3月笔试 |
| **考试科目** | 公共科目（公基+行测）+ 专业科目 |
| **福利待遇** | 参照军官待遇，有编制 |
| **竞争程度** | 逐年上升，但低于国考 |

## 考试内容

### 公共科目
- 基本知识（政治、经济、法律、人文、科技）
- 岗位能力（言语、数量、判断、资料分析）

### 专业科目
根据报考岗位而定，共13类44个专业
